Cloud Engineer Demo

Design • Deploy • Manage Cloud Systems

Job Summary

Design, implement, and manage scalable, secure, and highly available cloud infrastructure across AWS, Azure, or Google Cloud Platform.

Key Responsibilities

  • Design infrastructure using Infrastructure as Code (IaC)
  • Manage cloud resources (compute, storage, networking)
  • Implement CI/CD pipelines
  • Monitor system performance (Prometheus, Grafana)
  • Ensure security compliance (IAM, encryption)
  • Troubleshoot system issues

Required Skills

  • AWS / Azure / GCP
  • Docker & Kubernetes
  • Python, Bash scripting
  • Networking (VPC, Subnet, Load Balancer)
  • Linux / Unix

Tools & Technologies

AWS Docker Kubernetes Terraform Git Jenkins Prometheus

Soft Skills

  • Problem solving
  • Communication
  • Teamwork (Agile/Scrum)
